# Sprint 3: Categories, Overdue, Navigation & Weather Enhancements

## Goals
- Add custom color-coded categories for tasks
- Overdue task highlighting (distinct color for incomplete overdue tasks)
- Month and year navigation views for quick date jumps
- Refine localStorage structure for categories and tasks
- UI polish for clarity and minimalism
- Allow user to set location (lat/lon) for weather, with fallback to browser geolocation
- Cache weather grid endpoint for efficiency

## Deliverables
- Category creation, editing, color selection, and assignment to tasks
- Overdue incomplete tasks highlighted distinctly
- Month and year navigation overlays/views
- Data model and localStorage updates for categories
- Improved UI for category and overdue states
- User can set or update location for weather (with geolocation fallback)
- Weather API grid endpoint cached and refreshed as needed

## Stretch Goals
- Category filtering in timeline view
- Animations for navigation transitions

## Out of Scope
- Authentication, cloud sync, advanced calendar features
