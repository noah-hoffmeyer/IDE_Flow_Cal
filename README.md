# IDE Flow Calendar

A timeline-first todo and calendar application built with React + Vite.

## Repository Layout

- `app/` — Frontend application source code and build configuration.
- `docs/setup.md` — Quick onboarding and local development steps.
- `docs/technical/architecture.md` — Technical architecture and data model notes.
- `docs/roadmap/` — Sprint planning and release milestone documents.
- `vercel.json` — Vercel deployment configuration.

## Quick Start

```bash
cd app
npm install
npm run dev
```

Then open `http://localhost:5173`.

## Scripts

From `app/`:

- `npm run dev` — Start local development server.
- `npm run build` — Create production build.
- `npm run preview` — Preview production build locally.
