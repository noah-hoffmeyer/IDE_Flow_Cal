# Sprint 1: Core Timeline, Tasks & Weather Setup

## Goals
- Establish project structure and basic UI scaffolding
- Implement horizontal scrolling timeline of days as the main view
- Add ability to create, edit, and delete tasks for a specific day
- Store tasks in localStorage
- Basic Today button to jump to current date
- Set up weather API integration (NWS) and fetch forecast for today + next few days
- Display simple weather icon (sun/cloud/rain) for today and next few days

## Deliverables
- Minimal web app with horizontal timeline (no month/year nav yet)
- Task creation, editing, deletion for a single day
- Tasks persist in localStorage
- Today button functional
- Weather API fetch logic and display of weather icons for today/next days
- Clean, minimal UI skeleton

## Stretch Goals
- Basic task completion toggle
- Simple drag-and-drop for untimed tasks (optional, if time)

## Out of Scope
- Categories, color-coding, overdue highlighting, month/year nav, advanced sorting
