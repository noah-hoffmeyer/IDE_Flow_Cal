# Sprint 2: Task Management, Timeline & Weather Polish

## Goals
- Add support for timed and untimed tasks
- Timed tasks auto-sort by time; untimed tasks are manually ordered (drag-and-drop)
- Mark tasks as complete/incomplete; completed tasks appear greyed out
- Move tasks across days via drag-and-drop
- Improve timeline UI/UX (smooth scrolling, visual polish)
- Refine weather icon display (handle API errors, loading states, minimal icons)

## Deliverables
- Timed/untimed task distinction in UI and logic
- Auto-sorting for timed tasks
- Manual ordering for untimed tasks (drag-and-drop)
- Task completion toggle and visual state
- Drag-and-drop to move tasks between days
- Timeline UI improvements (scrolling, highlighting today, etc.)
- Weather icons reliably shown for today and next few days

## Stretch Goals
- Keyboard navigation for timeline
- Simple overdue highlighting (if time)

## Out of Scope
- Categories, color-coding, month/year navigation, advanced overdue logic
